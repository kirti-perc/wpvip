A Booking Plugin that book a restaurant table with specific date and time with number of persons.

== Description ==
admin can add holidays, week day off, start time and end time of the booking day.



Features:

1. booking with multistep
2. booking with food and without food 
3. mail notification
4. manage booking in admin side




== Installation ==
1. Upload "quick-table-booking" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
